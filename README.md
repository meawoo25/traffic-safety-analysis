# 🚦 Traffic Accident Analysis & Prediction (Machine Learning Project)

This project analyzes traffic accident data and builds a machine learning model to predict the likelihood of accidents based on various factors such as weather, road conditions, and time.

---

## 🚀 Project Highlights

* Analyzed real-world traffic accident dataset
* Performed data cleaning and preprocessing
* Conducted Exploratory Data Analysis (EDA)
* Built classification models to predict accident occurrence
* Identified key factors contributing to accidents

---

## 📂 Dataset

* Traffic accident dataset containing road and environmental conditions

### Features include:

* Time of accident
* Weather conditions
* Road type
* Light conditions
* Vehicle type
* Location

### 🎯 Target:

* Accident occurrence (Yes/No) OR severity level

---

## 🧠 Technologies Used

* Python
* Pandas & NumPy
* Matplotlib & Seaborn
* Scikit-learn

---

## ⚙️ Workflow

### 1. Data Preprocessing

* Handled missing values
* Encoded categorical variables
* Converted time-based features
* Removed outliers

### 2. Exploratory Data Analysis (EDA)

* Accident distribution over time
* Impact of weather conditions
* Road type vs accident frequency
* Visualization using graphs

### 3. Feature Engineering

* Extracted useful time features (hour, day)
* Encoded categorical features

### 4. Model Building

* Logistic Regression
* Decision Tree
* Random Forest Classifier

### 5. Model Evaluation

* Accuracy
* Precision
* Recall
* F1 Score

---

## 📊 Results

* Random Forest achieved the best performance
* Weather and time were key factors influencing accidents
* Model successfully predicts accident likelihood
